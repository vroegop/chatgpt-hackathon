# Welkom to the ChatGPT Hackathon!

## Where do I start?
*Install Node, Python3 or Java*

*Add an API key to your desired stack: https://platform.openai.com/account/api-keys*

* **HTML:**
  > cd html
  > 
  > npx live-server
* **NodeJS:**
  > cd nodejs
  > 
  > node index.mjs
* **Python:**
  > cd python
  >
  > python3 index.py
* **Java:**
  > cd java
  >
  > javac -d out *.java && java -cp out Index
