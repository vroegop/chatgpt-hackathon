# Welkom op de Accelerate ChatGPT Hackathon!

## Hoe werkt het?
*Disclaimer: je moet uiteraard Node (recente versie) of Python3 installeren!*

* **HTML:**
  > cd html
  > 
  > npx live-server
* **NodeJS:**
  > cd nodejs
  > 
  > node index.mjs
* **Python:**
  > cd python
  >
  > python3 index.py
* **Java:**
  > cd java
  >
  > javac -d out *.java && java -cp out Index
